Now

This is a /now page. It's a place to share what I'm focused on at this point in my life.

Updated February 2026

Things I'm thinking about

  • • Valence data (aka reaction data, vote data, or polislike data)
  • • How we can use valence data to make sense of our collective reaction to content, to create maps of human perspectives
  • • How we can not just depolarize culture, but use new tools to help stabilize as much difference as possible in culture
  • • Boundaries as sites of tension, the resolution of which supports human progress
  • • How we can use machine learning to make sense of our collective reactions to information (statements), leaving the interpretation of language to humans themselves

Current projects

  • • Google Maps-inspired interfaces for exploring high-dimensional valence data — Perspective Map Explorer v2
  • • R&D infrastructure for more people to participate in finding best practices for working with valence data and their resulting maps — valency-anndata

Near future projects

  • • Interfaces for gathering high-dimensional reaction data to content (YouTube videos, public lectures, live events)
  • • Agent skills for making exploration and human cartography possible with minimal coding skills
